==========================================
LIBRARY DETAILS & PURPOSE
==========================================

EXPO ECOSYSTEM
---------------
All Expo packages provide mobile-specific functionality:
- expo: Core SDK providing unified APIs across iOS, Android, and Web
- expo-router: File-based routing system (replaces React Navigation)
- expo-auth-session: OAuth flows for mobile authentication
- expo-secure-store: Encrypted storage for sensitive data (tokens, passwords)
- expo-location: GPS and location services access
- expo-maps: Native map components (uses Google Maps on native platforms)
- expo-file-system: Read/write files to device storage
- expo-document-picker: Native file picker dialogs
- expo-image-picker: Access device camera and photo gallery
- expo-sharing: Share content with other installed apps
- expo-constants: Access app version, build info, etc.
- expo-linking: Handle deep links and universal links
- expo-status-bar: Control status bar appearance and style
- expo-web-browser: Open URLs in in-app browser

REACT NATIVE FRAMEWORK
-----------------------
- react-native: Core React Native framework for building native mobile apps
- react-native-web: Enables React Native components to work on web platform
- react-native-screens: Native screen components for better performance
- react-native-safe-area-context: Handle notches, status bars, and safe areas
- react-native-gesture-handler: Declarative gesture API for touch interactions
- react-native-reanimated: High-performance animation library using native driver
- react-native-worklets: JavaScript functions that run on UI thread

REACT NATIVE UI COMPONENTS
---------------------------
- react-native-calendars: Calendar views and date pickers for mobile
- react-native-signature-canvas: Signature pad with drawing support
- react-native-webview: Render web content in native WebView

REACT NATIVE UTILITIES
-----------------------
- @react-native-async-storage/async-storage: Persistent key-value storage
- @react-native-community/datetimepicker: Native date/time picker dialogs

UI COMPONENTS
-------------
- @expo/vector-icons: Comprehensive icon library with thousands of icons

DATA MANAGEMENT
---------------
- @tanstack/react-query: Advanced data fetching with automatic caching,
  background updates, optimistic updates, and error handling
  (Mobile uses React Query, web uses SWR)

DEVELOPMENT TOOLS
-----------------
- @types/react-native: TypeScript type definitions for React Native
- cross-env: Set environment variables cross-platform (Windows/Mac/Linux)
